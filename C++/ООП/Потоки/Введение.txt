Р. Лаффоре ст 536

Потоки в С++ представляют собой объекты классов, представляющие собой монитор, клавиатуру, файлы и пр. предназначенное для ввода/вывода на экран или файлы. Заметим, что библиотека IOSTREAM, отвечающая за ввод/вывод информации является производной от других объектов (подробнее см ст 538 Р Лафоре). 

Для потоков существуют, флаги формотирования, которые позволяют формотрировать входящие в них данные, вот некоторые из них:
skipws Пропуск пробелов при вводе
left Выравнивание по левому краю
right Выравнивание по правому краю
internal Заполнение между знаком или основанием числа и самим числом
dec Перевод в десятичную форму
oct Перевод в восьмеричную форму
hex Перевод в шестнадцатеричную форму
boolalpha Перевод логического 0 и 1 соответственно в «false» и «true»

устанавливаются они так: std::имя_потока.setf(std::ios::имя_флага) (снять флаг: .unsetf) 

Причём существуют и манипуляторы (определённые флаги формотирования), которые можно установить, просто бросив в поток например: std::cout << hex << var -выовод var в 16-ичном формате 
////(подробнее о флагах и манипуляторах: Р.Лаффоре ст 539)

Функции
Также для потоков, помимо setf/usetf существуют другие функции (хотя правилнее назвать методы), спомощью которых можно устанавливать флаги для поределённого потока, вот некоторые из них: 

ch = fill(); Возвращает символ заполнения (символ, которым заполняется неиспользуемая часть текстового поля; по умолчанию пробел)
fill(ch); Устанавливает символ заполнения
p = precision(); Возвращает значение точности (число выводимых знаков для формата с плавающей запятой) precision(p); Устанавливает точность р
w = width(); Возвращает текущее значение ширины поля (в символах)
width(w); Устанавливает ширину текущего поля
setf(flags); Устанавливает флаг форматирования (например, ios::left)
unsetf(flags); Сбрасывает указанный флаг форматирования
setf(flags, field); Очищает поле, затем устанавливает флаги форматирования

устанавливают их так: std::имя_потока.имя_функции(необходимые аргументы)
