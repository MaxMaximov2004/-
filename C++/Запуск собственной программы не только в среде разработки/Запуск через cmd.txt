Бугагашеньки, теперь поговорим о том как можно запускать свою программу и передаватьей переменные НЕ ЧЕРЕЗ среду разработки и проч. а точнее мы можем передать программе некоторые данные до её непосредственной работы, для этого необходимо
1) открыть коммандную строку и вбить в ней полный путь до нашей прграммы + парамтры через пробел (не вплотную к адресу программы)
в этом случае мы передаём так называемые аргументы коммандной строки в нашу программу, которые затем использует наша прграмма ( по средствам счётчика-argc и списка аргументов-argv)

код:
#include <iostream>
#include <string>

int main(int argc, char* argv[])
{

    std::cout << "argumrnts of main:\n";

    for (int i = 0; i < argc; i++) {
        std::cout << i << ") " << argv[i] << std::endl;
    }

    return(0);
}
